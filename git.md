Depois de ajeitar o SSH:

1. Add origin
```
git remote add origin git@github.com:GabrielCordeiroBarrosoTeles/Contatinhos-php.git
```

2. Configurar usuario
```
git config --global user.email "gabrielpikachuoficial@gmail.com"
```

3. Pronto o sorvetinho
```
git add .

git commit -m "mensagem"

git push origin main
```
